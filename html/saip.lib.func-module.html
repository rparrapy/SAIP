forma_ciclo(nodo, nodos_explorados=[], aristas_exploradas=[], band=False, nivel=1)

source code 

Determina recursivamente si existe un ciclo en la componente conexa del grafo de items a la que pertenece un item dado.

Parameters:
  • nodo ({Item}) - Item dado.(Normalmente acaba de añadírsele un relación)
  • nodos_explorados (list({Item})) - Lista de items que ya han sido visitados.
  • aristas_exploradas (list({Relacion})) - Lista de relaciones visitadas.
  • band (Bool) - Bandera que indica si ya se ha encontrado un bucle.
  • nivel (Integer) - Indica la cantidad de llamadas recursivas anidadas realizadas.
Returns: Bool
True si existe un bucle, False en caso contrario.
