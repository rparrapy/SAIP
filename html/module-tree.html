      • saip.lib.app_globals: The application's Globals object
      • saip.lib.auth: Módulo que provee los predicates checkers para controlar la autorización de los usuarios para el acceso a los recursos del sistema.
      • saip.lib.base: The base Controller API.
      • saip.lib.func: Módulo que provee funciones varias para la utilización en los controladores.
      • saip.lib.helpers: WebHelpers used in SAIP.
    • saip.model: The application's model objects
      • saip.model.app: Módulo que define las clases del modelo del sistema no relacionadas a la autentificación ni a la autorización.
      • saip.model.auth: Módulo que define las clases del modelo del sistema relacionadas a la autentificación y a la autorización.
    • saip.templates: Templates package for the application.
    • saip.tests: Unit and functional test suite for SAIP.
      • saip.tests.models: Unit test suite for the models of the application.
        • saip.tests.models.test_proyecto
        • saip.tests.models.test_rol
        • saip.tests.models.test_usuario
    • saip.websetup: Setup the SAIP application
      • saip.websetup.bootstrap: Setup the SAIP application
      • saip.websetup.schema: Setup the SAIP application
